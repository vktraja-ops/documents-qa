# Documents Q&A

A RAG (Retrieval-Augmented Generation) app that lets you upload documents
and ask questions about them using Google Gemini + ChromaDB.

## Setup

1. Clone the repo:
   git clone https://github.com/vktraja-ops/documents-qa.git

2. Create a virtual environment:
   python -m venv .venv
   .venv\Scripts\activate

3. Install dependencies:
   pip install -r requirements.txt

4. Create a `.env` file with your API key:
   GOOGLE_API_KEY=your_key_here

5. Run the app:
   streamlit run c:/AI/Capstone_Project/query_documents_agenticAI.py
